## Cores

| Token | Hex | Uso |
|---|---|---|
| `primary` | `#0A2540` | Azul-marinho — cor principal, headers, botões primários |
| `success` | `#00A86B` | Verde-esmeralda — lucros, pagamentos, acções positivas |
| `danger` | `#E63946` | Vermelho — dívidas, faturas vencidas, alertas críticos |
| `warning` | `#F59E0B` | Amarelo-âmbar — alertas médios, stock baixo |
| `info` | `#3B82F6` | Azul — informações neutras |
| `surface` | `#FFFFFF` | Fundo dos cartões |
| `background` | `#F4F6F8` | Fundo principal da app |
| `text` | `#1A1A1A` | Texto principal |
| `text-secondary` | `#6B7280` | Texto secundário, labels |
| `border` | `#E5E7EB` | Linhas, separadores |

## Tipografia

- **Família:** Inter (Google Fonts, gratuita)
- **Pesos usados:** 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
- **Fallback:** Helvetica, Arial, sans-serif

## Logos

- `logo.svg` — versão horizontal (cabeçalho web, faturas PDF)
- `icon.svg` — só o "D" no quadrado (ícone Android/favicon)
